## Demo

1. Clone this repo → `git clone https://github.com/huangjulu/mr-transcript.git`
2. Open Chrome → go to `chrome://extensions`
3. Toggle **Developer mode** on (top right)
4. Click **Load unpacked** → select the `extension/` folder
5. Done — Mr. Transcript is now pinned in your toolbar

## What it does

Paste a YouTube URL. Pick a language. Hit download. That's it.

- **TXT** — clean text, ready to read or feed to AI
- **SRT** — timed subtitles, drag into any video editor
- **50+ languages** — flags included, because why not
- **Auto-detect** — on YouTube? URL is already there
- **Timestamps** — on or off, your call

## How it works (the short version)

YouTube doesn't want you downloading transcripts. Mr. Transcript doesn't care.

It runs inside the YouTube page itself, asks politely for the captions, and hands them to you as a file. YouTube thinks it's talking to itself.
